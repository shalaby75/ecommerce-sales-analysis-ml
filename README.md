# E-Commerce Sales & Profit Analysis with Machine Learning

## Project Overview

This project focuses on analyzing E-Commerce business data to understand sales performance, customer behavior, profits, and business trends using Data Analysis and Machine Learning techniques.

The project includes:

* Data Cleaning & Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Business Insights Extraction
* Machine Learning Model Training & Evaluation
* Data Visualization

The dataset was analyzed to identify top-performing categories, profitable states, monthly sales trends, and customer spending behavior.

# Dataset Information

The project uses real-world E-Commerce datasets containing:

* Customer Information
* Orders Information
* Product Categories
* Sales & Profit Data
* Sales Targets

The datasets were merged and cleaned to create a final dataset ready for analysis and Machine Learning.

---

# Data Cleaning

The following preprocessing steps were applied:

* Removed missing values
* Removed duplicate rows
* Converted date columns to datetime format
* Cleaned text columns using string operations
* Validated final dataset quality

---

# Feature Engineering

Created new features including:

* Month Extraction
* Profit Margin
* Average Item Price
* Customer Total Spending
* High Profit Flag

---

# Exploratory Data Analysis (EDA)

Performed multiple visual analyses including:

* Sales by Category
* Profit by Category
* Monthly Sales Trend
* Top States by Sales

Key Insights:

* Electronics achieved the highest sales.
* Clothing generated the highest profits.
* January had the highest monthly sales.
* Madhya Pradesh and Maharashtra were the top-performing states.

---

# Machine Learning

Built and evaluated Machine Learning models to predict profits.

Models Used:

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

Evaluation Metrics:

* MAE
* MSE
* R² Score

Linear Regression achieved the best overall performance for this dataset.
